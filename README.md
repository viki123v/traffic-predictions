# Предвидување на безбедност на делници

Точка 1 од анализата е обработена во `feature_importance.ipynb`

Точка 2 од анализата е обработена во `bounds.ipynb`

Точка 3 од анализата е обработена во `regressors.ipynb`

Точки 4 и 5 од анализата се обработени преку `main.py`

За оклучување на веб апликација треба да се влезе во api фолдерот и да се изврши командата `uvicorn main:app --reload`.
Потоа во прелистувач да се оклучи `http://localhost:8000`
